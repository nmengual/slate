    • Introducción

    Bienvenido a la documentación para la conexión mediante Restful a la tienda de Edisofer.

    Para comenzar cualquier comunicación con la api, es necesario disponer de un token. En la sección autenticación esta toda la información.

    Autenticación

    Para autenticarte y obtener el token:

    curl -X POST "https://edisofer.xper-enti.com/rest/V1/integration/admin/token"
  -F username=usuario
  -F password=contraseña

    La respuesta es un token con esta estructura:

    "29kambbq1ddrmufmm1r1owmmybwzbp91"

    Edisofer usa API keys para acceder a la API.

    Edisofer espera que la API key este incluida en todas las peticiones al servidor:

    Authorization: token

    Debes reemplazar token con tu API key.

    Productos

    Añadir nuevo producto

    curl -X POST \
  https://edisofer.xper-enti.com/rest/V1/products/ \
  -H 'Accept: application/json' \
  -H 'Authorization: Bearer token' \
  -H 'Cache-Control: no-cache' \
  -H 'Content-Type: application/json' \
  -d '{
    "product": {
        "sku": "123456",
        "name": "Producto de prueba",
        "price": 10.00,
        "type_id": "simple",
        "attribute_set_id": 4,
    "custom_attributes": [
        {
            "attribute_code": "category_ids",
            "value": [1, 2, 3]
        }
    ]
    }
}'

    HTTP Request

    POST https://edisofer.xper-enti.com/rest/V1/products

    Campos de "product"

    Campo Tipo Descripción
    sku string ISBN del producto
    name string El nombre del producto
    price float El precio del producto
    type_id string "simple" por defecto
    attribute_set_id id 4 por defecto
    custom_attributes array Atributos personalizados (1)

    (1) Campos de "custom_attributes"

    Todos los valores del array de custom_attributes se componen de la siguiente estructura:

    Campo Tipo Descripción
    attribute_code string Código del attributo
    value multi Valor del atributo

    Editar un producto

    Para editar el precio de un producto con sku/ISBN 123456

    curl -X PUT \
  https://edisofer.xper-enti.com/rest/all/V1/products/123456 \
  -H 'Accept: application/json' \
  -H 'Authorization: Bearer token' \
  -H 'Cache-Control: no-cache' \
  -H 'Content-Type: application/json' \
  -d '{
    "product": {
        "price": 15.00,
    }
}'

    La petición devuelve el siguiente JSON:

    true

    HTTP Request

    PUT https://edisofer.xper-enti.com/rest/all/V1/products/{sku}

    Eliminar un producto

    Para eliminar el producto con el sku/ISBN 123456

    curl -X DELETE \
  https://edisofer.xper-enti.com/rest/V1/products/123456 \
  -H 'Authorization: Bearer token' \
  -H 'Cache-Control: no-cache' \

    La petición devuelve el siguiente JSON:

    true

    HTTP Request

    DELETE https://edisofer.xper-enti.com/rest/V1/products/{sku}

    Obtener categoría por nombre

  'https://edisofer.xper-enti.com/rest/V1/categories/list?searchCriteria[filterGroups][0][filters][0][field]=name&searchCriteria[filterGroups][0][filters][0][value]=aeronautica' \
  -H 'Accept: application/json' \
  -H 'Authorization: Bearer token' \
  -H 'Cache-Control: no-cache' \

    Errores

    La API de edisofer tiene los siguientes códigos de error:

    Código de error Descripción
    400 Bad Request -- Petición incorrecta
    401 Unauthorized -- No tienes permisos para acceder al recurso
    404 Not Found -- No se encuentra el recurso
